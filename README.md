# hyprtsuki

Dotfiles for a Hyprland setup. Minimal, and feel like it's not doing too much.

I’m not actively maintaining this, so use at your own risk.

# requirements
- Arch Linux (or Arch-based distro) i guess you could use another distro.. but i haven't tested.
- Hyprland
- Git

# dependencies
```bash
sudo pacman -S hyprpaper hyprland rofi fastfetch fish
```

# installation
```bash
git clone https://github.com/Apzhyn/hyprtsuki ~/.config/
```
this might overwrite your current config files so be careful
